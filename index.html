ℹ️ Permissions & Browser Compatibility

  • 🔒 HTTPS required for: Geolocation, Camera/Mic, Clipboard (write), Wake Lock, Notifications, Service Worker, Speech Recognition, Web Share.
  • 📱 Mobile-only: Vibration (Android), Device Orientation (iOS needs permission prompt).
  • 🌐 Chromium-only: Battery Status API, Network Information API, Speech Recognition.
  • 🦊 Firefox / Safari: Some APIs have limited or no support — the badge next to each card reflects the detected state in your current browser.
  • 🏠 This page is deployed on GitHub Pages (HTTPS), so all permission-gated APIs will work as long as your browser supports them.

Built with pure HTML · CSS · Vanilla JS — no frameworks, no build tools.
